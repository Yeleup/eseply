# Модуль: Типы абонентов

## Терминология

Тип абонента в коде называется `ClientType`.

Значение хранится в поле клиента `client_type` и в поле тарифа `client_type`.

## Назначение

Тип абонента определяет, какой тариф применяется к клиенту внутри выбранной tenant-организации.

## Доступные значения

- Физ. лицо;
- ИП;
- ТОО;
- Коммерческие объекты;
- Бюджет.

## Правила

`ClientType` является enum, а не редактируемым справочником.

Отдельный CRUD-раздел для типов абонентов не используется.

Тип абонента выбирается в карточке клиента.

Тариф выбирается по услуге организации, типу абонента и периоду.

Если нужен другой набор типов абонентов, это отдельное изменение бизнес-правил и enum.
