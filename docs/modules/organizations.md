# Модуль организаций

## Назначение

Модуль хранит данные организации, от имени которой ведётся учёт коммунальных услуг.

В административной панели Filament организация используется как tenant-область.

## Терминология

Tenant-организация в коде называется `Organization`.

## Основные поля

- название организации;
- БИН / ИИН;
- телефон;
- адрес;
- банк;
- IBAN;
- примечание.

## Правила доступа

- пользователь может быть участником нескольких организаций;
- пользователь может открыть только организацию, к которой он привязан;
- при создании организации через Filament текущий пользователь автоматически привязывается к созданной организации;
- переключение между организациями выполняется через tenant menu Filament.

## Связи

- организация имеет много пользователей;
- организация имеет много клиентов;
- организация имеет много коммунальных услуг;
- организация имеет много категорий тарифов;
- организация имеет много тарифов;
- организация имеет много нормативов;
- пользователь имеет много организаций.

## Ограничения

Тип начисления не относится к организации и хранится только в карточке абонента.
