# Модуль: Категории тарифов

## Терминология

Категория тарифа в коде называется `TariffCategory`.

Таблица категорий тарифов называется `tariff_categories`.

## Назначение

Модуль хранит категории, по которым выбирается тариф для клиента.

Примеры категорий:

- население;
- юрлица;
- коммерческие объекты.

## Основные поля

| Поле | Описание |
|---|---|
| Организация | Tenant-организация, поле `organization_id` |
| Название | Название категории тарифа, поле `name` |
| Статус | Активная / неактивная, поле `status` |
| Примечание | Дополнительная информация, поле `note` |

## Правила

Каждая tenant-организация ведёт свои категории тарифов.

Категория тарифа выбирается в карточке клиента.

Категория тарифа используется для выбора тарифа при начислениях.

Категории тарифов могут повторяться в разных организациях.

## Связи

Категория тарифа используется в тарифах `Tariff`.

Категория тарифа используется в нормативах `Normative`.

## Административная панель

Категории тарифов управляются через Filament resource `TariffCategoryResource` внутри выбранной tenant-организации.

В навигации админки раздел называется «Категории тарифов» и расположен после «Абоненты».

Форма категории тарифа должна позволять редактировать название, статус и примечание.

В списке категорий тарифов должны отображаться название, статус и дата создания.
